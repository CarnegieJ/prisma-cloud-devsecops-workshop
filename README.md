# Cloud DevSecOps Workshop with Cortex Cloud, HCP Terraform and GCP
cortex-cloud-devsecops-workshop
DevSecOps with Cortex Cloud
Workshop date: March 5, 2026

Learning Objectives:
- Gain an understanding of DevSecOps and Infrastructure as Code (IaC) concepts, specifically using Terraform as the IaC framework.
- Scan Infrastructure as Code files for security misconfigurations locally, using Checkov to identify risks before deployment.
- Set up CI/CD pipelines that automate security scanning and policy enforcement, integrating tools such as GitHub, HCP Terraform, and Cortex Cloud.
- Fix security findings and Google Cloud Platform resource misconfigurations, leveraging Cortex Cloud to remediate issues detected in IaC and cloud configurations.
